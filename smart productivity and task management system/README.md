🚀 Smart Productivity & Task Management System

This is a premium full-stack task management application designed to help you manage your day-to-day tasks and track your productivity efficiently.

🧐 What is this project about? (Project Goal)

We all handle multiple tasks daily, but tracking progress and deciding priorities can be difficult.
This system provides a dashboard with productivity analytics and gives smart recommendations on what to focus on next.

🛠️ Tech Stack

Frontend: HTML5, Modern CSS (Premium Glassmorphism Design), JavaScript

Backend (API): Python Flask (Full-scale REST API framework)

Database: SQLite (with SQLAlchemy ORM)

Authentication: JWT (JSON Web Tokens) & Bcrypt hashing (for security)

Charts: Chart.js (for graphical visualizations)

✨ Key Features
🚨 1. Smart Authentication

User Registration & Login

Secure password hashing

JWT Authorization to protect user data and sessions

📝 2. Efficient Task Management

Create, edit, and delete tasks

Priority Settings: High, Medium, Low

Deadlines for each task

Status tracking (Pending → Completed)

📊 3. Powerful Dashboard & Analytics

Visual Charts: Weekly productivity and task distribution

Smart Recommendations: Suggestions like “Focus on high-priority tasks first”

Real-time Stats: Total, completed, pending, and overdue tasks

⚙️ 4. Advanced Settings

Single Page View (no page reloads)

Profile updates and user preferences
