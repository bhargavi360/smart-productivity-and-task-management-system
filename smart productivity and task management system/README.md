# 🚀 Smart Productivity & Task Management System

Idhi oka **premium full-stack task management application**. Meeru mee day-to-day tasks ni manage chesukovadaniki, mee productivity ni track cheyadaniki idhi chala "thopu" ga pani chesthundhi.

---

## 🧐 Asalu idhi dheni gurinchi? (Project Goal)
Manam daily chala tasks chestham, kani mana progress entha undhi? Ee task mundhu cheyali? Ee details telusukovadam kastam. 
Ee system mee dashboard lo mee productivity analytics ni chupisthundhi, meeru edi focus cheyali ane "Smart Recommendations" kuda isthundhi.

---

## 🛠️ Tech Stack (Manam use chesina Technology)
- **Frontend**: HTML5, Modern CSS (Premium Glassmorphism Design), JavaScript.
- **Backend (API)**: **Python Flask** (Full-scale REST API framework).
- **Database**: SQLite (SQLAlchemy ORM use chesam).
- **Authentication**: JWT (JSON Web Tokens) & Bcrypt hashing (Security kosam).
- **Charts**: Chart.js (Graphical visuals kosam).

---

## ✨ Key Features (Mukhyamaina Features)

### 🚨 1. Smart Authentication
- User Registration & Login.
- Password hashing (Mee passwords chala secure ga untayi).
- **JWT Authorization**: Evadu padithe vadu mee data ni access cheyakunda JWT secure sessions untayi.

### 📝 2. Efficient Task Management
- Create, Edit, and Delete tasks.
- **Priority Settings**: High, Medium, or Low priority set cheyochu.
- **Deadlines**: Task eppudu poorthi avvalo date set chesukovachu.
- Status update (Pending to Completed).

### 📊 3. Thopu Dashboard & Analytics
- **Visual Charts**: Mee weekly productivity graph and task distribution automatic ga update avthundhi.
- **Smart Recommendations**: Backend mee tasks ni analyze chesi "Focus on high priority tasks first" lanti suggestions isthundhi.
- **Real-time Stats**: Total, Completed, Pending, and Overdue tasks count instant ga kanipisthundhi.

### ⚙️ 4. Advanced Settings
- Single Page View switching (Reload lekunda view marthundhi).
- Profile info update and preferences management.

## 🚀 How to Run? (Ela run cheyali?)

### 🐍 1. Backend Start cheyandi
Terminal open chesi `backend` folder ki velli:
```bash
cd backend
python3 -m pip install -r requirements.txt
python3 app.py
```
App `http://127.0.0.1:5000` lo run avthundhi.

### 🌐 2. Frontend Open cheyandi
`frontend` folder lo unna `index.html` file ni mee browser (Chrome/Edge) lo open cheyandi.
